Welcome to pylibressl's documentation!
======================================

PyLibreSSL -- Python3 bindings to LibreSSL using CFFI. It aims to provide
interface to the most important bits of LibreSSL on Python 3.4+. Supported
platforms: GNU/Linux, Windows (coming soon).

For installation instructions see
`README.md <https://korobok.stonehenge-3.net.ru/yl3dy/pylibressl/src/master/README.md>`_.
