Welcome to pylibressl's documentation!
======================================

PyLibreSSL -- Python3 bindings to LibreSSL using CFFI. It aims to provide an interface to the most important bits of LibreSSL in Python 3.4+. Supported platforms: GNU/Linux, Windows. It is distributed under the terms of the `MIT license <https://github.com/yl3dy/pylibressl/blob/master/LICENSE>`_.
